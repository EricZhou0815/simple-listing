# Simple Listing

这是一个简单的二手物品 listing 网站。

你可以用它来：

- 在首页展示要出售的物品
- 点进每个物品查看详情
- 用管理员账号登录后台
- 在后台新增、修改、删除物品

这个项目适合没有编程知识的业主使用。下面的步骤尽量写得非常简单，你只需要照着做就可以。

## 你需要先准备什么

你需要一台电脑，并且电脑上已经安装好以下两个东西：

- `Node.js`
- `npm`

如果你不确定有没有安装，可以先请帮你搭建这个项目的人帮你确认。

## 第一步：打开这个项目

1. 打开电脑里的终端或命令行工具。
2. 进入这个项目文件夹。
3. 确认你现在在 `simple-listing` 目录里。

如果你不懂怎么进入文件夹，可以直接告诉帮你安装的人，让他帮你打开这个项目。

## 第二步：安装依赖

在终端里输入：

```bash
npm install
```

等待它执行完成。

这一步的作用是安装网站运行所需要的文件。

## 第三步：启动网站

在终端里输入：

```bash
npm run dev
```

启动成功后，终端会显示一个本地地址，通常是：

```bash
http://localhost:3000
```

## 第四步：打开首页

1. 打开浏览器。
2. 在地址栏输入 `http://localhost:3000`
3. 按回车。

你会看到网站首页，里面展示了各种二手物品。

## 第五步：查看物品详情

1. 在首页点击任意一个物品图片或标题。
2. 进入详情页后，可以看到这个物品的照片、价格、地点和介绍。

## 第六步：进入后台管理

1. 在网页右上角点击“管理后台”。
2. 输入管理员账号和密码。

登录后，你就可以管理物品了。

### 管理员账号密码怎么设置

这个项目现在从环境变量读取管理员账号密码。

本地开发时，在项目根目录创建 `.env.local`，写入：

```bash
ADMIN_USERNAME=your_username
ADMIN_PASSWORD=your_password
```

部署到 Vercel 时，在 Project Settings 里添加同名环境变量即可。

### GitHub Actions 部署到 Vercel

如果你想用 GitHub Actions 自动部署，需要在 GitHub 仓库里设置这些 Secrets：

- `VERCEL_TOKEN`
- `VERCEL_ORG_ID`
- `VERCEL_PROJECT_ID`

然后把代码 push 到 `main` 分支，工作流会自动部署到 Vercel。

同时，在 Vercel 的 Environment Variables 里设置：

- `ADMIN_USERNAME`
- `ADMIN_PASSWORD`
- `SELLER_MESSENGER_URL`

如果你在本地跑，也可以在 `.env.local` 里写同样的变量。

### 页面标题怎么改

在后台的 `Homepage Text` 区域里，可以直接修改 `Page title`。

改完后，浏览器标签页标题会同步更新。

### Talk to seller 按钮怎么配

在部署环境里设置：

```bash
SELLER_MESSENGER_URL=https://m.me/your.page.username
```

或者使用 Messenger 页面链接：

```bash
SELLER_MESSENGER_URL=https://www.facebook.com/messages/t/PAGE_ID
```

设置后，商品详情页会显示 `Talk to seller` 按钮，点击会在新标签页打开 Messenger。

## 第七步：添加新物品

1. 进入后台后，左侧会看到“添加物品”。
2. 依次填写：
   - 名称
   - 价格
   - 分类
   - 成色
   - 地点
   - 照片链接
   - 详情
3. 点击“添加物品”按钮。

添加完成后，新物品会出现在列表里。

## 第八步：修改物品

1. 在后台列表里找到要修改的物品。
2. 点击“修改”。
3. 左侧表单会自动填入这个物品的信息。
4. 改好以后点击“保存修改”。

## 第九步：删除物品

1. 在后台列表里找到不需要的物品。
2. 点击“删除”。

删除后，这个物品就不会再显示。

## 第十步：退出后台

1. 点击后台页面右上角的“退出”。
2. 你会回到未登录状态。

## 停止网站

如果你想关掉网站：

1. 回到终端窗口。
2. 按 `Ctrl + C`

这样网站就会停止。

## 常见问题

### 1. 浏览器打不开网页怎么办？

先确认终端里的网站还在运行。

如果终端已经关掉了，重新输入：

```bash
npm run dev
```

### 2. 登录不了后台怎么办？

请确认你已经设置了：

- `ADMIN_USERNAME`
- `ADMIN_PASSWORD`

### 3. 物品图片显示不出来怎么办？

请检查你填写的照片链接是否正确。

如果你不想自己找图片链接，可以先让技术人员帮你改成支持直接上传图片。

### 4. 我关掉电脑后数据还在吗？

当前这个版本把数据保存在浏览器本地。

也就是说：

- 在同一台电脑、同一个浏览器里，数据一般会保留
- 如果换电脑、换浏览器，数据可能看不到

如果你希望数据长期保存，建议下一步接数据库和后台管理服务。
